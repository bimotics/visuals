But as dashboard guru Steven Few has frequently pointed out, needle gauge indicators are not an efficient use of screen real estate. A more efficient way to display a value within an expected range and indicate a target value is to use a simple horizontal bullet bar.

In the second example the chart type is bar horizontal (bhs) and there are four chart markers (chm) within the bar. The first marker is red, the second is yellow and the third is green.

The first marker is the red bar that extends 50% of the bar's distance
The second marker is the yellow line that extends from .5 to .75
The third marker the green line that extends from .75 to the end of the bar
We can also draw a short vertical black bar to indicate our target value of .81
Finally, the overall height of the entire bar is controlled by the chart bar height parameter chbh=10
